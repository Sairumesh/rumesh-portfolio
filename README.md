# Rumesh Sree K G — Portfolio

This is my personal portfolio site. I built it to showcase projects, data work, and writing with a dark, animated feel and a small 3D touch.

## What’s inside
- React + TypeScript + Vite
- Tailwind with a custom token set; shadcn/ui components for consistency
- Framer Motion for micro-interactions; React Three Fiber for lightweight 3D
- React Router for page routing

## Running it
```bash
npm install
npm run dev
npm run build
```
The resume download points to `public/assets/resume.pdf`. Social links live in `src/components/SocialLinks.tsx`.

## Notes for future tweaks
- Colors and typography: adjust `src/index.css` tokens and `tailwind.config.ts`.
- SEO/meta: update `<title>` and meta tags in `index.html`.
- Content: edit the section data in `src/pages/*` (projects, skills, articles, coding profiles).
